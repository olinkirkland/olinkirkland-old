About Me

My name is Olin. I'm a software developer living and working in Cologne, Germany.

My favorite projects involve games and UI gamification. I excel best in fast paced environments.

I primarily use Actionscript and Flex, but picking up a new framework or language isn't a problem. I'm dedicated to developing software that adheres to best practices and consistent code structure.

My work is pretty varied: I've made applications for museum exhibits, developed front-end solutions for security system management, and currently work on mobile games in Cologne.

Outside of my work I like traveling, hiking, developing side-projects, and playing strategy games online with friends.
